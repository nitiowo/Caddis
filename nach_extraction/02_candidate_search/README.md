# Candidate Search Scripts

- `run_makeblastdb.sh`: build BLAST databases from transcriptome assemblies
- `run_blastn.sh`: search transcriptomes with nucleotide BLAST
- `run_tblastx.sh`: search transcriptomes with translated nucleotide BLAST
- `run_tblastn.sh`: search transcriptomes with translated protein queries
- `arrayblast.job`: scheduler job file for array-based BLAST searches
