# COI Phylogeography Scripts

- `build_coi_dataset.R`: prepares the COI alignment, distance matrix, species summaries, and locality summary tables
- `coi_rooted_tree.R`: builds the rooted COI tree with limnephilid outgroups
- `lf_haplotype_network.R`: generates LF haplotype network plots and summary tables
- `lf_mantel.R`: tests locality-level genetic distance against geographic and toxicity differences
